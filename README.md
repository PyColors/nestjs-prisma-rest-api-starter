

## Description
NestJS and Prisma starter.


A REST API starter that makes an incredibly productive stack for rapidly building well-structured, type-safe, and maintainable backend applications.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
